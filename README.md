# Project Overview
This project is an end-to-end Sales Data Analysis performed using Python. The goal of this project is to clean, transform, and analyze sales data to extract meaningful business insights and understand patterns in customer behavior, product performance, and regional sales distribution.

## Tools & Libraries Used
Python 
Pandas (Data Cleaning & Transformation)
NumPy (Data Handling)
Matplotlib (Data Visualization)
Seaborn (Statistical Visualization)

## Workflow of the Project
1️ Data Cleaning & Preparation (Pandas)
Handled missing values
Removed unnecessary columns
Converted data types where required
Performed grouping and aggregation for analysis
2️ Exploratory Data Analysis (EDA)
Performed in-depth analysis using visualizations to understand:
Customer demographics
Product category performance
State-wise and occupation-wise sales
Gender and age group distribution
Overall revenue patterns

## Visualizations Used
The following charts were used to extract insights:
📈 Bar Charts (for comparisons like State vs Sales, Product Category vs Sales)
📉 Line Charts (for trend visualization)
🍩 Pie Charts (for part-to-whole analysis)
📊 Count Plots (for categorical distributions)
📦 Horizontal Bar Charts (for better category readability)

🔍 Key Insights
Certain states contribute significantly higher sales compared to others.
Specific product categories dominate overall revenue.
Customers from particular occupations show higher purchasing behavior.
Age group and gender distribution help understand target audience segments.
Sales patterns vary significantly across regions and demographics.
